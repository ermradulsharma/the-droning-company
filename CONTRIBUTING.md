# Contributing to The Droning Company

First off, thank you for considering contributing to **The Droning Company**! It's people like you that make this platform a great resource for the drone community.

## ü§ù How Can I Contribute?

### Reporting Bugs

If you find a bug, please use our [Bug Report Template](.github/ISSUE_TEMPLATE/bug_report.md). Be sure to include:

- A clear description of the issue.
- Steps to reproduce.
- Expected vs. actual behavior.
- Screenshots if applicable.

### Suggesting Enhancements

Have an idea? Great! Please use our [Feature Request Template](.github/ISSUE_TEMPLATE/feature_request.md) to explain your suggestion.

### Pull Requests

1. **Fork** the repository.
2. **Create a branch** for your feature or fix (`git checkout -b feature/amazing-feature`).
3. **Commit** your changes (`git commit -m 'Add amazing feature'`).
4. **Push** to the branch (`git push origin feature/amazing-feature`).
5. **Open a Pull Request** using our PR template.

- Use descriptive commit messages.
- Reference issues where applicable (e.g., `Fixes #123`).
