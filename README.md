# The Droning Company - Frontend

Welcome to the frontend repository for **The Droning Company**, a platform dedicated to drone pilots and aerial services. This project is built using Next.js and React, providing a robust interface for job listings, pilot profiles, and community engagement.

## ğŸ› ï¸ Getting Started

### Prerequisites

Ensure you have Node.js and npm (or yarn) installed on your system.

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd the-droning-company-frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the results.

### Building for Production

To create an optimized production build:

```bash
npm run build
npm start
```
